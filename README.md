# MingStore
<strong>项目</strong>：JavaWeb商城 (B2C)<br>
<strong>介绍</strong>：这是我学习框架前的第一个Web项目，jsp+servlet+mysql 搭建。<br>
<strong>功能</strong>：实现了响应式(bootstrap)前台页面，邮箱验证登录，浏览商品，全站搜索，查看历史记录，购物车详情，订单查看,在线支付(易宝接口),后台商品上架，下架管理等主要功能。<br>
